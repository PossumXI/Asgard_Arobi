# Demo Output Directory

This directory contains demo recordings generated by the ASGARD Demo Recorder.

## Contents

- `manifest.json` - Metadata about all recordings
- `screenshots/` - PNG screenshots captured during demo
- `videos/` - WebM video recordings of each system

## Regenerating

To regenerate demos, run from the `scripts` directory:

```powershell
npm run demo
```

## Cleaning

To remove all recordings:

```powershell
npm run clean
```
