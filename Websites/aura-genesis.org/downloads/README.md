## Verification

To verify package integrity, compare the SHA256 checksum of your downloaded file with the values above.

### Windows
```cmd
certutil -hashfile filename.exe SHA256
```

### macOS
```bash
shasum -a 256 filename.dmg
```

### Linux
```bash
sha256sum filename.AppImage
```
