## Image Specifications

### Social Media Images
- **Dimensions**: 1200x630 pixels (1.91:1 aspect ratio)
- **Format**: JPG (preferred) or PNG
- **File Size**: Under 5MB
- **Background**: Dark theme matching site design
- **Text**: High contrast white text on dark background
- **Branding**: Include Aura Genesis branding

### Logo Specifications
- **Primary Logo**: SVG format preferred, PNG fallback
- **Square Format**: 512x512px minimum
- **Transparent Background**: PNG format
- **Vector Format**: Scalable SVG for different sizes

## Creation Guidelines

1. Use the HTML template files as reference for design
2. Maintain brand colors: Black (#000), White (#FFF), Green accents (#00ff88)
3. Include key messaging and visual elements
4. Ensure text is readable at small sizes
5. Test on actual social media platforms

## Usage

These images are referenced in the meta tags of each page:
- Open Graph tags for Facebook/LinkedIn
- Twitter Card tags for Twitter
- SEO and sharing optimization

## Tools for Creation

- Canva, Figma, or Adobe Photoshop
- GIMP (free alternative)
- Online design tools like Looka or Hatchful

## Testing

After creating images, test them using:
- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)
