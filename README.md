# PANDORA (ASGARD)

Planetary-scale autonomous systems architecture for orbital sensing, humanoid
robotics, orchestration, and cybersecurity. This repository is organized as a
monorepo with strict directory ownership.

## Core Directories

- `Silenus` - Satellite program and orbital perception.
- `Sat_Net` - Delay tolerant networking layer.
- `Hunoid` - Humanoid robotics systems.
- `Nysus` - Central orchestration and MCP services.
- `Giru` - Security, red/blue teaming, and defensive automation.
- `Data` - Databases, migrations, and edge sync.
- `Control_net` - Kubernetes and infrastructure management.
- `Hubs` - Streaming and viewing interfaces.
- `Websites` - Public and government web portals.
- `Documentation` - Generated and build-time documentation.
